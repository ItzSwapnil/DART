

# **GUIDELINES FOR PROJECT REPORT – SEMESTER VII**

This document specifies the standard format and structure for preparing the Semester VII Project Report. Students are required to follow these guidelines strictly for uniformity and academic quality.

## **Formatting Style**

Paper size: A4

Font: Cambria

Font size: 12 for text, 14 (bold) for chapter titles (CHAPTER TITLES SHOULD BE IN UPPERCASE ONLY), 13 (bold) for chapter subtitles.

Line spacing: 1.5

Margins: Left 1.5 inches, Right 1 inch, Top 1 inch, Bottom 1 inch

Page number: Bottom center

Text should be justified.

## **Structure of the Frontmatter**

Cover Page (TEMPLATE PROVIDED)

Declaration

Acknowledgement

Abstract (INCLUDING KEYWORDS)

Table of Contents

List of Figures (if applicable)

List of Tables (if applicable)

## **Structure of the Main Content**

Chapters – INTRODUCTION, LITERATURE REVIEW, OBJECTIVES, METHODOLOGY (as applicable), NUMERICAL / EXPERIMENTAL RESULTS (as applicable)

Future work (to be completed in Semester VIII)

References

Appendices (if any)

### **NOTE:**

1. The Frontmatter should be printed on the right side only (as a single page print). The Main Content should be printed in double sided format with each chapter starting on the right sided page.
2. A plagiarism report (for both actual and AI content) must be submitted, signed by the assigned supervisor.
